# ALICE-Swarm

ALICEエコシステム向けのマルチエージェント群制御ライブラリ。Boidsアルゴリズム、フォーメーション制御、コンセンサスプロトコル、タスク割り当て、障害物回避、群メトリクスを純Rustで提供。

## 機能

- **Boidsアルゴリズム** -- 分離・整列・結合(半径・重みを設定可能)
- **フォーメーション制御** -- ライン、リング、V字、グリッド隊形とフォーメーションステアリング
- **コンセンサスプロトコル** -- 任意の通信トポロジー上でのスカラー/ベクトルコンセンサス
- **通信トポロジー** -- 完全結合、リング、スター、k近傍グラフ
- **タスク割り当て** -- 貪欲法とオークションベースのエージェントへのタスク割り当て
- **障害物回避** -- 斥力ポテンシャル場、経路遮断検出
- **リーダー-フォロワー** -- フォロー距離とゲイン設定可能なリーダー追従
- **群メトリクス** -- 重心、平均速度、広がり、秩序パラメータ、最小距離、直径、衝突数、接続性

## アーキテクチャ

```
Agent (位置, 速度, 最大速度, 最大力)
    |
    +-- Boids (分離 + 整列 + 結合)
    +-- Formation (ライン / リング / V字 / グリッド)
    +-- LeaderFollower (リーダー追従)
    |
    v
Topology (完全結合 / リング / スター / k-NN)
    |
    v
Consensus (スカラー / ベクトル収束)
    |
    v
Task Allocation (貪欲法 / オークション)
    |
    v
障害物回避 --> 群メトリクス
```
